prva zad
da se napise klasa User vo koja se cuva
    int id;
    string email;
    string name;
    string surname;
    int connections;
vo klasata da se napise konstruktor, operator -- koj ako connections se poce od 0 gi namaluva za 1, operator >=, i da se overload operatorot za pecatenje.

da se napise klasa UserData vo koja se cuva dinamicki alocirana niza od User i nejzinata golemina

Da se napista potrebnite konstruktori i destruktor

da se overload operator [], i ako nema user so takov index da se frli isklucok od klasata UserWithIndexNotFound

da se overload operator +=
da se napise funkcija filterByConnections, koja prima broj na connections, i vraca nov objekt od UserData so site 
clenovi so tolku connections

da se overload operator za pecatenje

da se napise globalna funkcija za citanje od file User ReadUser (ifstream &in).
