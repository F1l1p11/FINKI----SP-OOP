Da se napise klasa User vo koja se cuvaat id celobrojna promenliva, email string promenliva, name string promenliva, surname string promenliva
connections celobrojna promenliva. Za klasata da se napisat potrebnite konstruktori.
Da se preoptovari operatorot -- vo prefix notacija, koj go namaluva brojot na promenlivata connections za 1 dokolku promenlivata ima vrednost pogolema od 0.
Da se preoptovari operatorot >= so koj se sporeduva dva objekti od klasata user spored brojot na connections.
Da se preoptovari operatorot <<. (da se vidat test case za kako se pecati)


Da se napise klasa UserData vo koja se cuva dinamicki alocirana niza od User klasata, i celobrojna promenliva vo koja se cuva brojot na elementi vo nizata.
Da se napisat potrebnite konstruktori i destruktor.
Da se preoptovari operatorot [] koj prima cel broj za indeks, koj frla isklucok od klasata UserWithIndexNotFound dokolku ne postoi user so toj indeks vo nizata.
Da se preoptovari operatorot +=, za dodavanje nov element od klasata user.
Da se napise metod koj prima broj na konekcii, i vraca nova noza so objekti cii broj na konekcii e ist so brojot na konekcii prosleden vo metodot.
Da se preoptovari operatorot za pecatenje, koj ako nizata nema elementi pecati poraka "UserData is empty!". (da se vidat test kejsoj za pecatnenje)
Da se napise globalna funkcija so ime ReadUser, so koja se citaat elementi od klasata user, i se vraca objekt od klasata user.
Vo main na potrebnitewe mesta da se spravi so isklucocite.
